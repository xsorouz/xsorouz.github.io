Projets

Chaque fiche : contexte → action → résultat (+ stack). Synthétique, orienté impact.

Référentiel clients unifié

Contexte : 12 sources CRM/e-commerce.
Action : matching + dédoublonnage, règles qualité, traçabilité.
Résultat : +18% de segmentation exploitable.
Stack : dbt, Spark, PostgreSQL.

Orchestration & monitoring

Contexte : pipelines quotidiens multi-équipes.
Action : Airflow/Kestra, tests, métriques Prometheus, alerting.
Résultat : −60% d’incidents mensuels.
Stack : Airflow, Kestra, Prometheus, Grafana, Python.

Dashboards pilotage

Contexte : KPIs hétérogènes et non partagés.
Action : cadrage KPIs, modélisation sémantique, PBI/Tableau.
Résultat : adoption ↑, décision ↗, temps de préparation ↓.
Stack : Power BI, Tableau, SQL, dbt.

Scores & prévisions

Contexte : ciblage marketing peu fiable.
Action : features simples + modèles explicables + doc.
Résultat : ROI campagnes ↑, prévisions plus stables.
Stack : Python (scikit-learn), FastAPI, Streamlit.
